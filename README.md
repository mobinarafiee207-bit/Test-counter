# 📝 Test Counter

An interactive exam tool for tracking correct and wrong answers.

---

## ✨ Features

- ✅ Two separate counters for correct and wrong answers
- 📊 Real-time percentage calculation
- 💾 Auto-save with LocalStorage
- 📋 One-click result report
- ⌨️ Keyboard shortcuts (↑↓R)
- 📱 Fully responsive design

---

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage API

---

## 🚀 How to Use

1. Click `+` to add a correct answer
2. Click `-` to add a wrong answer
3. Press `↑` key for correct, `↓` key for wrong
4. Press `R` to reset everything
5. Click `Save Result` to see the summary report
